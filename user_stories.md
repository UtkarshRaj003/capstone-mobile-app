# Capstone Project - User Stories

### 1. User Registration (Signup)
As a new user, I want to create an account using my username, email, and password so that I can securely access the application and maintain my personal data.

### 2. User Authentication (Login)
As a registered user, I want to log into the application using my email and password so that I can access my personalized dashboard and saved content.

### 3. Dynamic Home Screen
As a user, I want to see a well-structured home screen with the app's brand logo in the header and a clean layout showing available items/categories.

### 4. Detailed Item View
As a user, I want to tap on an item from the home screen and navigate to a detailed screen so that I can view specific details, descriptions, and information about that item.

### 5. Local Data Persistence (Favorites)
As a user, I want to mark specific items as "Favorites" and save them to local storage so that my preference is saved even if I close and reopen the app.

### 6. External API Integration
As a user, I want the application to fetch and display real-time data from an external API so that I can see up-to-date and dynamic information on the dashboard.

### 7. Settings Menu Navigation
As a user, I want a clearly visible settings icon in the app header/drawer that opens up a list of available configuration menus.

### 8. App Configuration & Theme Toggling
As a user, I want a dedicated settings screen where I can configure my application preferences, such as switching between Light and Dark modes.

### 9. Local/Push Test Notifications
As a user, I want to receive a triggered alert or local notification reminder on my device when a specific task is updated or a test notification button is tapped.
